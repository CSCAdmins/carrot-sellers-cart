# Contributing to Carrot Sellers

Thank you for your interest in contributing to the Carrot Sellers UFO/UAP Claims Tracker. This project aims to maintain objective, evidence-based documentation of statements made by prominent UFO/UAP talking heads.

## ü•ï Our Mission

We evaluate public figures based on five key criteria:
- **Evidence Quality**: Source credibility and verification
- **Logical Consistency**: Consistency of positions over time
- **Transparency**: Willingness to cite sources and correct errors
- **Sensationalism**: Use of dramatic language vs. measured presentation
- **Monetization Ethics**: Transparency about conflicts of interest

## üîß How to Contribute

### For Quote Submissions or Corrections

1. **Fork the repository**
2. **Create a branch**: `git checkout -b add-quote-person-name-date`
3. **Make your changes** to the appropriate person's markdown file
4. **Follow the quote format**:
   ```markdown
   <div class="quote-box">
       <div class="quote-date">Month Day, Year</div>
       <div class="quote-source">Source: Specific Source Name</div>
       <div class="quote-text">
           Exact quote here in quotation marks
       </div>
       <div class="quote-context">
           <div class="quote-context-label">Context:</div>
           Additional context about the situation
       </div>
       <div class="fact-check [true|false|unverified]">
           <div class="fact-check-label">Fact Check: [TRUE|FALSE|UNVERIFIED]</div>
           Explanation of verification status
       </div>
   </div>
   ```
5. **Submit a Pull Request** with:
   - Clear title describing the addition
   - Link to original source
   - Brief explanation of significance

### For New People (After Issue Approval)

1. **Wait for Issue approval** before starting
2. **Use the person template** from `docs/people/example-person.md`
3. **Include all required sections**:
   - Header with photo and basic info
   - Carrot meter breakdown with detailed scoring
   - At least 3 documented quotes with sources
   - Updates & corrections section
   - Sources & references section
4. **Add to people index** (`docs/people/index.md`)
5. **Include high-quality profile photo** (300x300px minimum)

## üìè Quality Standards

### Sources Must Be:
- **Publicly available** (audio, video, articles, social media)
- **Directly attributable** to the person
- **Timestamped and dated** when possible
- **Archived** (use archive.org links when available)

### Quotes Must Be:
- **Verbatim** (no paraphrasing)
- **In context** (include relevant surrounding context)
- **Significant** (notable claims, not casual conversation)
- **Verifiable** (traceable to original source)

### Scoring Must Be:
- **Evidence-based** with specific examples
- **Consistent** with existing rubrics
- **Documented** with reasoning for each criterion
- **Updated** only with new evidence

## üö´ What We Don't Accept

- **Personal attacks** or character assassination
- **Speculation** without evidence
- **Off-topic content** not related to UFO/UAP claims
- **Promotional content** or self-promotion
- **Copyright violations** (fair use excerpts only)
- **Doxxing** or private information
- **Political content** unrelated to UFO/UAP claims

## üîç Review Process

### For Regular Contributions:
1. **Automatic checks** (formatting, links)
2. **Maintainer review** (accuracy, sources)
3. **Merge** if approved
4. **Feedback** if changes needed

### For New People:
1. **Community discussion** on Issue (7+ days)
2. **Maintainer evaluation** against inclusion criteria
3. **PR creation** only after approval
4. **Standard review process** for PR

### For Score Changes:
1. **Evidence review** by multiple maintainers
2. **Community input** period
3. **Transparent decision** with documentation
4. **Update notification** to affected parties
